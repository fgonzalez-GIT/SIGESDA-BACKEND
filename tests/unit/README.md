# Unit Tests

Este directorio contiene tests unitarios para servicios, repositories y helpers.

## Estructura recomendada:

```
tests/unit/
├── services/
│   ├── cuota.service.test.ts
│   ├── recibo.service.test.ts
│   └── ...
├── repositories/
│   ├── cuota.repository.test.ts
│   ├── recibo.repository.test.ts
│   └── ...
└── utils/
    ├── parentesco.helper.test.ts
    └── persona.helper.test.ts
```
